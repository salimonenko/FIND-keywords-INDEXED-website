Открытие программы индексирования файлов сайта производится путем открытия файла 
INDEXER_files/file_INDEXER.php
(по ссылке в браузере, как обычно).

1. Необходимо проиндексировать ИМЕНА всех файлов сайта (нажав кнопку).
2. Необходимо проиндексировать СОДЕРЖИМОЕ файлов сайта (нажав кнопку).
3. После этого можно осуществлять поиск по ключевым словам (кнопка "Слово поиск").

Для обеспечения функциональности кнопок запуска открытия файлов необходим установленный текстовый редактор Notepad++ 
(операционная система Windows). Если Notepad++ не будет установлен, основная функциональность сохранится, но лог-файлы
придется открывать вручную. В некоторых операционных системах Windows, возможно, потребуется скорректировать команду
запуска этого редактора.
