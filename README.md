# FIND-keywords-INDEXED-website
Quick search for keywords on a website through indexing.

The program indexes website files. Index directories and files are created in the "metaphones" directory. After indexing the NAMES of all (non-prohibited) files, and after indexing the CONTENT of these files, you can search by keywords using logical expressions (may be  && || operators). Searching is very fast, in a fraction of a second. The program has been tested with PHP 5.3 and PHP 8.0. It is cross-browser compatible, including Firefox 24, 35, and Edge.

Программа делает индексирование файлов сайта. Индексные каталоги и файлы создаются в каталоге metaphones. После индексирования ИМЕН всех (не запрещенных) файлов, после индексирования КОНТЕНТА этих файлов можно проводить поиск по ключевым словам. используя логические выражения (операторы &&  ||). Поиск производится очень быстро, за доли секунды. Проверена работа программы в PHP 5.3, PHP 8.0. Кроссбраузерно, проверена работа в браузерах Firefox 24, 35; Edge.
